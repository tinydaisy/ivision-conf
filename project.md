# iVision Conf — Описание проекта

## Настройка контента конференции через JSON

Весь контент страницы хранится в файле `apps/tma/ivision-7.json`.
Для новой конференции — создай новый JSON файл и новый HTML (или параметризуй имя файла через URL).

### Настраиваемые поля JSON

| Поле | Описание |
|------|----------|
| `name` | Название конференции (отображается мелко над заголовком) |
| `name_sub` | Подзаголовок крупным шрифтом (БИЗНЕСЫ ВЛИЯНИЯ...) |
| `tagline` | Описание под заголовком |
| `slogan` | Слоган — отображается в разделе Миссии в рамке |
| `dates` | Даты проведения |
| `format` | Формат (Онлайн-конференция для предпринимателей) |
| `hero_label` | Pill-тег в начале hero (напр. «Онлайн-конференция») |
| `capacity_total` | **Настраивается** — общее кол-во мест (напр. `"1000"`) |
| `capacity_free` | **Настраивается** — осталось бесплатных мест (напр. `"800"`) |
| `stats` | Массив цифр-счётчиков на hero. Поле `suffix` для `%` |
| `marquee_facts` | **Настраивается** — факты для бегущей строки под hero |
| `mission.photo_url` | Фото в разделе миссии (`/img/mission.jpg`) |
| `bonuses.items` | Подарки за регистрацию (топ-3 показываются в hero) |
| `tariffs[].gc_script_id` | ID виджета GetCourse для тарифа |
| `tariffs[].gc_script_src` | URL скрипта GetCourse для тарифа |



## Что это за проект

iVision Conf — это Telegram Mini App (TMA) для управления онлайн-конференциями. Участник открывает приложение прямо из Telegram — без скачивания и отдельной регистрации. Платформа объединяет программу конференции, B2B-маркетплейс, розыгрыш призов и реферальную игру в едином интерфейсе.

## Для кого

**Участники конференции** — предприниматели, эксперты и специалисты из СНГ, аудитория iVision. Основной канал — Telegram, устройство — мобильный телефон.

**Организаторы** — команда iVision, управляют контентом через закрытую веб-панель (`/admin`).

## Страницы, которые планируются

Приложение имеет **3 состояния** в зависимости от статуса участника.

### Состояние 1 — Незарегистрированный
- Лендинг с кнопкой перехода на GetCourse

### Состояние 2 — Во время конференции (основное)

Нижний таббар с 4 вкладками:

| Вкладка | Что внутри |
|---------|-----------|
| **Программа** | Карусель спикеров, переключатель дней, карточки сессий, профиль спикера, LIVE-баннер с таймером, дашборд |
| **Услуги** | B2B-маркетплейс: заявки на выступление, консалтинг, менторство, разработка |
| **Розыгрыш** | Баланс билетов, просмотр билетов, список призов |
| **Игра** | Реферальная программа: генерация ссылки, статистика, прогресс до следующего уровня призов |

Вкладки «Розыгрыш» и «Игра» заблокированы до подписки на каналы организатора (+ каналы спикеров опционально). При разблокировке выдаётся +1 билет.

### Состояние 3 — После конференции

| Вкладка | Что внутри |
|---------|-----------|
| **Календарь** | Нетворкинг-события, мастермайнды, будущие конференции |
| **Услуги** | Аналогично состоянию 2 |
| **Материалы** | Архивы прошлых конференций, «Главный урок» (воронка) |
| **Игра** | Реферальная программа продолжается |

### Дополнительно
- **Экран блокировки** — подписка на каналы
- **Лок-квест «Свой человек»** — ироничная форма/квиз перед входом в игру
- **Веб-панель `/admin`** — управление спикерами, программой, рассылками, настройками

## Стиль дизайна

- **Цвета:** градиент логотипа фиолетово-синий `#6c63ff` → голубой `#3ecfcf`
- **Тема:** адаптируется к светлой/тёмной теме Telegram через CSS-переменные `--tg-theme-*`
- **Шрифт:** системные шрифты (SF Pro, Segoe UI) для нативного ощущения
- **Макет:** mobile-first, нижний таббар, прогрессивное раскрытие деталей по тапу
- **Тон:** профессиональный, но с иронией — прохождение «квеста» подаётся как «проверка на своего»
- **UX-принципы:** не более 4 вкладок, скелетон-загрузчики, хаптик-фидбэк, счётчики в реальном времени
